# 🚀 AI 智慧投資組合優化器：全算法邏輯與代碼手冊 (v3.4)

本手冊定義了系統中「如何評分 (Evaluation)」與「如何搜索 (Optimization)」的兩大核心技術模組。所有核心計算邏輯已遷移至 WebAssembly (WASM) 以確保極致效能與公式私密性。

---

## 第一部分：搜索算法 (Optimization Algorithms)

這決定了系統如何遍歷成千上萬種權重組合。

### 1. 🎲 蒙地卡羅模擬 (Monte Carlo Simulation)
*   **邏輯**：隨機在 N 維空間中抽取樣本。這是一種「暴力」搜索法，優點是簡單且不會陷入局部最優解。
*   **適用場景**：初步探索解空間。

### 2. 🧬 遺傳算法 (Genetic Algorithm) - 【推薦】
*   **邏輯**：模擬生物進化。保留最優組合（精英制），透過「錦標賽選擇」父代，進行「交叉 (Crossover)」與「變異 (Mutation)」。
*   **關鍵機制**：
    *   **精英保留 (Elitism)**：前 5% 最優組合直接進入下一代。
    *   **變異 (Mutation)**：隨機交換兩隻股票權重或重置權重，跳出局部最優點。

### 3. 🧱 網格搜索 (Grid Search - 5% 步進)
*   **邏輯**：將權重空間離散化為 5% 的倍數。適合尋找整數權重的最佳組合。

## 第二部分：核心評分算法 (Evaluation Models)

### 1. 🤖 超級 AI 優化 v2.0 (六邊形戰士) - `super_ai_v2`
*   **設計哲學**：引入幾何乘積 (Geometric Product) 與自適應門檻。強迫系統挑選出在六維空間（回報、回撤、波動、勝率、平滑度、尾部風險）都沒有短板的組合。
*   **門檻過濾 (Gatekeeping)**：
    *   MaxDD > 25% ➔ 淘汰 (Score: -9000)
    *   年化收益 < 5% ➔ 淘汰 (Score: -9001)
    *   平滑度 (R²) < 0.85 ➔ 淘汰 (Score: -9003)
*   **評分公式**：
    `Score = (Return^0.4) * (Risk^0.3) * (Stability^0.2) * (Channel^0.1) * 100`
    *   `Risk` 包含 **CVaR Penalty** (尾部風險懲罰)。

### 2. 💎 極致穩定 v2 (雙向波動通道) - `ultra_smooth`
*   **設計哲學**：基於對數回歸殘差分析。假設最完美的資金曲線應該是一條對數空間下的直線。
*   **核心指標**：Symmetric Ulcer Index (對稱潰瘍指數)。
*   **熔斷機制**：單期波動超過 **8%** ➔ 評分為 0。
*   **通道一致性**：計算偏離趨勢線 3% 以內的數據點比例。

### 3. 💎 極致穩定 v1 (類定存效果) - `ultra_smooth_v1`
*   **設計哲學**：將「增長穩定性」與「回撤承受度」進行非線性平方懲罰。
*   **公式**：`(Smoothness^2) * ((1 - MaxDD)^2) * WinRate`

### 4. 🤖 超級 AI 優化 v1 (基礎模式) - `default`
*   **邏輯**：線性加權評分。
*   **公式**：`(Sharpe * 0.6) + (Calmar * 0.2) + (Smoothness * 0.2)`

## 第三部分：技術指標工具箱 (Math Toolbox)

### 📈 平滑度 (Smoothness / R-Squared)
衡量資金曲線與對數趨勢線的貼合程度。

### 🛡️ 尾部風險懲罰 (CVaR Penalty)
針對歷史數據中極端不利的 5% 事件進行額外扣分。
`Penalty = exp(CVaR * 10)`

### 📉 通道得分 (Channel Score)
基於最大偏離值 (Max Deviation) 的指數衰減評分。
